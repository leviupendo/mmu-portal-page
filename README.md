# MMU Student Portal Clone

## About This Project

This project was built purely for fun. I was curious about how web interfaces 
work and wanted to see if I could recreate the MMU Student Portal using 
just HTML and CSS. No tutorials, no course requirement — just curiosity 
and free time.

## What I Did

I took the MMU Student Portal interface and rebuilt it from scratch as a 
static web page. The project includes the full layout, styling, navigation 
and visual elements that make up the portal interface.

## Why I Did It

No big reason honestly. I was sitting around, opened VS Code and thought 
— let me see if I can build this. A few hours later it was done. Sometimes 
the best projects are the ones you build just because you felt like it.

## What I Learned

- How HTML structure works for large web pages
- How CSS handles layouts and styling
- How to use Bootstrap for responsive design
- How to push a project to GitHub from VS Code
- That even a "just for fun" project teaches you more than you expect

## Tools Used

- HTML
- CSS
- Bootstrap
- Font Awesome icons
- VS Code
- GitHub

## How to View It

Just download the files and open `MMU Student Portal.html` in any browser.
